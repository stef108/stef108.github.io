ICT?

                      Waarom heb ik ICT gekozen?

                      Ik heb altijd wel wat gehad met ICT, maar mijn keus voor ICT was niet zo sterk gemotiveerd als anderen.
                      Ik wist het niet zeker dat dit hem ging worden. Ik ben overgehaald door de meeloopdag.
                      En ik ben blij dat ik overgehaald ben want ik ben erachter gekomen dat ik ict gewoon heel leuk vind.

                      Maar waarom dan deze opleiding?

                      Zoals ik eerder zei werd ik overtuig door de meeloopdagen.
                      Deze studie geeft je meerdere uitgangpunten waar je naartoe kan werken,
                      hierdoor ben ik zelfverzekerd dat ik het ideale stuk van ICT voor mezelf kan kiezen.
